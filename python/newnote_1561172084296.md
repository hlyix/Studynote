#### rstrip()和strip() 
Python能够找出字符串开头和末尾多余的空白。要确保字符串末尾没有空白，可使用方法rstrip() 。 
```python
❶ >>> favorite_language = 'python '
❷ >>> favorite_language  
'python '
❸ >>> favorite_language.rstrip()  
'python'
```
你还可以剔除字符串开头的空白，或同时剔除字符串两端的空白。为此，可分别使用方法lstrip() 和strip() ： 
```
❹ >>> favorite_language = ' python ' ❷ >>> favorite_language.rstrip()  ' python'
❸ >>> favorite_language.lstrip()  'python ' ❹ >>> favorite_language.strip()  'python'
```
